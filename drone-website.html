03 / Principle

HOW IT WORKS

01
FLIGHT STABILIZATION
The flight controller continuously reads IMU data (gyroscope + accelerometer) and applies PID control to all four motors, maintaining stable, level flight automatically.
02
GPS POSITIONING
The GPS module transmits real-time latitude/longitude coordinates to the flight controller, enabling waypoint navigation, geofencing, and return-to-home functionality.
03
CONTINUOUS SENSOR SWEEP
LiDAR and ultrasonic sensors continuously scan the environment in all directions, measuring distances to surrounding objects and building a real-time obstacle map.
04
OBSTACLE DETECTION TRIGGER
When an obstacle is detected within the configured threshold distance, the system immediately triggers the avoidance algorithm and alerts the ground station dashboard.
05
PATH RE-PLANNING
The path planning algorithm calculates the optimal avoidance maneuver ‚Äî speed reduction, directional change, or altitude shift ‚Äî and executes the new trajectory smoothly.
06
MISSION CONTINUATION
After successfully avoiding the obstacle, the drone resumes navigation toward its target destination, resuming its original flight plan or calculating a new optimal path.

SENSOR FUSION

Combines data from multiple sensors (IMU, GPS, LiDAR) into a single accurate state estimate using Kalman filtering.
üó∫Ô∏è
PATH PLANNING
A* / RRT Algorithm
Finds the shortest collision-free path from current position to goal, dynamically re-planning when obstacles are encountered.

Advanced algorithm that builds a map of an unknown environment while simultaneously tracking the drone's position within it.
